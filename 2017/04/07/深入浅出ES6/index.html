const

    1 一旦申明必须赋值
    2和let一样，只能在所声明的块级作用域内有效。
    3也不能提升，存在暂时性死区。
    4不可重复声明
    5.const声明一个只读的常量，一旦声明，敞亮的值就不能改变。const pi = 3014.
    6.const 本质就是，当基本数值类型字符创，boolean ，数值，
    const  a=2，a是不可以改变的，但是数组和对象是可以变得。

总结let const clas生命的全局变量不属于顶层对象的的属性，而es5中顶层对象和全局属性是一个概念。

function move({x = 0, y = 0} = {}) {
return [x, y];
}

move({x: 3, y: 8}); // [3, 8]
move({x: 3}); // [3, 0]
move({}); // [0, 0]
move(); // [0, 0]
上面代码中，函数move的参数是一个对象，通过对这个对象进行解构，得到变量x和y的值。如果解构失败，x和y等于默认值。

注意，下面的写法会得到不一样的结果。

function move({x, y} = { x: 0, y: 0 }) {
return [x, y];
}

move({x: 3, y: 8}); // [3, 8]
move({x: 3}); // [3, undefined]
move({}); // [undefined, undefined]
move(); // [0, 0]
