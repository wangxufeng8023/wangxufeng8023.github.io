发表于 2017-04-10 | 分类于 vue.js | 阅读次数

前端大神尤雨溪给我们带来vue.js 比angular.js 轻量级，比React.js 更加简单，本文介绍简单Vue.js 基本语法。

遇见Vue.js

vue.js是一个前端框架，是一个MVVM框架，首先MVC和MVVM是一种设计模式，在在前端和后端都可以使用，比如java的spring3.0和struts2也是一种MVC框架，MVVM是
M 数据模型 VM(viewModel) 视图控制器 View 视图 ，vue.js的双向数据绑定是其精华部分:

vue 实例方法

组件树访问

vue实例提供了一些有用的属性和方法

$parent

用来访问当前组件实例的父实例

$root

$children

访问当前实例的直接子组件实例

$refs

用来访问使用v-ref指令的子组件

DOM访问

$el

用来访问挂载当前实例的DOM元素

$els

用来访问$el元素使用v-el指令的DOM

数据访问

$data

用来访问组件是里观察的数据对象

$options

用来访问组件实例化时的初始化选项对象

实例方法

$appendTo()

将el所指的DOM元素或片段插入到目标元素中
$appendTo(elementOrselector,callback)

$before()

将el所指的DOM元素插入到目标元素之前
$before(elementOrselector,callback)

$after()

将el所指的DOM元素插入到目标元素之后
$after(elementOrselector,callback)

$remove()

将el所指的DOM元素删除
$remove(callback)

$nextTick()

用来在下一次dom更新循环后执行指定的回调函数
$nextTick(callback)

demo

$on()

用来监听实例上的自定义事件
$on(event,callback) event 事件名

$once()

和 $on()的区别是只执行一次
$on(event,callback) event 事件名

$emit()

方法用来触发事件
$emit(event,args) args传递监听函数的参数

dispatch()

用来派发事件，现在当前实例触发，然后沿着父链一层一层向上，

$broadcast()

从来广播事件

$off()

用来删除事件的监听器
$off(event,callback) event 事件名
