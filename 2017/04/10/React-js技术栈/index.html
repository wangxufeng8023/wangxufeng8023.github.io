在顶层组件中将渲染的virtual Dom 通过render渲染到真实的Dom中去。

当组件初次渲染之后，再次更新时，React并不会把所有的组件重新渲染，他会用高效的dom diff算法进行局部渲染。

首次挂载组件时，

先执行getDefaultProps(),getInitialState(),componentWillMount,render 和componentDidMount方法

卸载时

执行 componentWillUnmount()

重新挂载组件的时候 getInitialState(),componentWillMount,render 和componentDidMount方法 不在执行getDefaultProps(),

再次渲染的时候，

组件接受更新的状态，componentWillReceiverProps shoulidComponentUpdate componentWillUpdate render componentDidUpdate
