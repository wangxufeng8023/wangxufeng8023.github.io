linux 到 windows

发表于 2017-05-05 | 分类于 系统安装 | 阅读次数

1 linux系统转window系统
今天将linux系统换成windows7 需要做一些特殊工作
1.首先做系统盘
1.1 可以选择Ultraiso 和大白菜等，我们选择的是ultraiso
1.2 下载win7.iso文件，把优盘刻录乘启动盘，当然ultraiso里面自带了win PE工具
1.3 优盘启动，启动后选择diskgenius分区的设置，这是linux系统的分区，我们要删除，同时建立windows的四个分区，
1.4 然后可以根据win PE工具的一键安装功能进行安装就可以了
2 安装双系统
安装双系统，必须先安装win7系统，然后在安装linux系统，否则会覆盖引导
2.1 右键点击我的电脑》管理》磁盘管理,选择一个空间较大的盘,右键选择压缩卷,笔者为Ubuntu14.04划分了50G的空间
2.2 同样是用当然ultraiso把优盘做成系统盘，然后u启动，进行安装即可
2.3 需要注意的是，安装类型时选择其他选项，不是ubuntu和win7共存的选项
2.4 选择空闲分区，就是我们新建的50G，然后新建分区,点击“+”,设置交换空间swap,笔者自己的内存是8G,也划分了8G,一般不大于物理内存就行，选择逻辑分区
2.5新建分区,点击“+”,设置Ubuntu启动引导区,一般划分200Mb足够了，挂载/boot
2.6 新建20G 分区，挂载点为/
2.7 剩下的空闲资源，设置挂载点为/home
2.8 在win7中安装easyBCD后打开,点击“Add New Entry”》选择Linux/BSD:具体设置如图,Type选择GRUB,Name自己随便写,笔者写的是Ubuntu作为标识,Drive:选取我们设置的/boot分区,有Linux标记,设置完成后点击“Add Entry”
